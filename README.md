# 🧱 SKBlock — Управление блоками и очистка миров

**SKBlock** — мощный инструмент для администраторов серверов Minecraft, позволяющий гибко настраивать права на установку/ломание конкретных блоков в разных мирах, а также организовывать автоматическую очистку выбранных типов блоков по расписанию.

---

## 🚀 Основные функции

* 🚫 **Запрет установки/разрушения**: Ограничивайте взаимодействие с конкретными блоками в определенных мирах.
* 🌍 **Миры-исключения**: Настройка белых и черных списков миров для работы плагина.
* 🧹 **Автоматическая очистка**: Удаление указанных блоков каждые 24 часа (или по вашему таймеру).
* ⚡ **Мгновенная очистка**: Команда для удаления лишних блоков прямо сейчас.
* 📊 **Гибкая настройка**: 3 отдельных файла для хранения списков блоков (`plase.yml`, `break.yml`, `delete.yml`).

---

## 🛠 Команды и Права

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/skblock plase add/rem` | Добавить/убрать запрет на **постановку** блока в руке | `skblock.admin` |
| `/skblock break add/rem` | Добавить/убрать запрет на **разрушение** блока в руке | `skblock.admin` |
| `/skblock delete add/rem` | Добавить/убрать блок в список **очистки** | `skblock.admin` |
| `/skblock delete time <сек>` | Установить интервал очистки | `skblock.admin` |
| `/skblock rightnow` | Запустить очистку миров **сейчас** | `skblock.admin` |

> **Примечание:** > * `skblock.break` — позволяет игроку ломать запрещенные блоки.
> * `skblock.plase` — позволяет игроку ставить запрещенные блоки.

---

## ⚙️ Конфигурация

Плагин автоматически создает необходимые файлы при первом запуске:

* **config.yml**: Основные настройки миров, сообщений и времени.
* **plase.yml**: Список ID блоков, которые нельзя ставить.
* **break.yml**: Список ID блоков, которые нельзя ломать.
* **delete.yml**: Список ID блоков для удаления.

---

## 📦 Установка

1. Скачайте собранный `.jar` файл.
2. Поместите его в папку `plugins` вашего сервера.
3. Убедитесь, что вы используете **Java 16** и версию сервера **1.16.5**.
4. Перезапустите сервер.
